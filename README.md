# ✨ HTML & CSS Effects and Patterns

A curated collection of **modern UI effects, animations, and visual patterns** built using **pure Vanilla HTML & CSS**.
This repository focuses on **visual polish, motion, and CSS mastery**, with zero external dependencies.

> 🎯 Goal: Demonstrate strong CSS fundamentals, creative visual thinking, and performance-friendly UI effects that can be reused in real-world frontend projects.

## 🛠️ Design & Engineering Principles

This repository follows **modern frontend best practices**:

* **CSS Variables** for centralized theming and easy customization
* **GPU-accelerated animations** using `transform` and `opacity`
* **Backdrop effects** (`backdrop-filter`) for depth and realism
* **Semantic HTML** for accessibility and maintainability
* **Zero dependencies** — no libraries, no frameworks

---

## 🧪 How to Use

1. Clone the repository
2. Open any `index.html` file in your browser
3. Customize styles using CSS variables defined in `style.css`

Each effect is **self-contained** and easy to integrate into real projects.
